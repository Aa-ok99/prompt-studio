# 🎨 Prompt Studio Master

คลัง Prompt Templates มืออาชีพ พร้อมฟีเจอร์จัดการระดับสูง สำหรับนักสร้างสรรค์ นักพัฒนา และนักการตลาด

## 🌐 ลิงก์ใช้งานเว็บ
- **GitHub Pages**: https://aa-ok99.github.io/prompt-studio/
- **GitHub Repo**: https://github.com/Aa-ok99/prompt-studio/

## ✨ ฟีเจอร์หลัก
- ✅ **90+ Prompt Templates** (60 เดิม + 30 ใหม่: Music, Video, Business)
- ✅ **ระบบตัวแปร (Variables)** - กรอกค่าอัตโนมัติ สร้าง Prompt พร้อมใช้
- ✅ **Favorites & Rating** - ปักหมุดและให้คะแนนเทมเพลตโปรด
- ✅ **จัดการเทมเพลตเองได้** - เพิ่ม/แก้ไข/ลบในหน้าเว็บ
- ✅ **วิเคราะห์ Prompt** - ตรวจสอบความชัดเจนและแนะนำปรับปรุง
- ✅ **Viewer** - อ่านไฟล์โค้ด/ข้อความ พร้อม Syntax Highlight
- ✅ **ประวัติ (History)** - บันทึก/นำเข้า/ส่งออก Prompt อัตโนมัติ
- ✅ **ทดสอบกับ AI ฟรี** - ลิงก์ 6 แพลตฟอร์ (Perplexity, Poe, You.com, Hugging Face, Character.AI, ChatSono)

## 🚀 วิธีใช้งาน本地
```bash
cd prompt-studio
python3 -m http.server 8080
```
เปิด `http://localhost:8080` ในเบราว์เซอร์

## 📁 โครงสร้างโปรเจกต์
```
prompt-studio/
├── index.html          # หน้าเว็บหลัก
├── style.css           # สไตล์ธีมมืด
├── README.md           # ไฟล์นี้
├── data/
│   └── templates.js   # 90+ Templates + ฟังก์ชันจัดการ
└── js/
    ├── app.js         # Logic หลัก (History, Import/Export)
    └── viewer.js      # ระบบดูไฟล์
```
